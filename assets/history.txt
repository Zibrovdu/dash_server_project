Версия 2.4 от 04.05.2021

1.	Вкладка «Информационные системы» - добавлена кнопка-ссылка на программу «Полномочия и роли пользователей МБУ»;
2.	Вкладка «Задачи / проекты»:
    -	Задачи с истекшим сроком (просроченные) выделится светло-розовым цветом. При этом можно самостоятельно изменить плановый срок выполнения задачи, чтобы задача не числилась в просроченных, или завершить ее;
    -	При заведении новой задачи или изменении существующий задачи на почту ответственным исполнителям (которые указаны в столбце «Исполнитель») будет приходить уведомление;
    -	В окне «изменение задачи/проекта» добавлен переключатель «задача выполнена». Теперь если необходимо поставить задаче статус выполнена необходимо перевести переключатель. При этом процент выполнения будет автоматически проставлен 100 %, а также будет доступно для изменения поле «фактическая дата выполнения». По умолчанию, в данном поле будет указана текущая дата, но ее можно будет самостоятельно поменять;
    -	Для каждой новой задачи будет автоматически проставляться дата постановки (заведения в систему) задачи. Изменять ее самостоятельно нельзя;
    -	Также для каждой задачи будет фиксироваться срок ее выполнения (в днях), который рассчитывается как разница между датой заведения задачи в системе и датой фактического выполнения задачи.

Версия 2.0 от 02.04.2021

1. Реализован новый функционал: Теперь можно добавлять и редактировать проекты и задачи прямо в дашборде.

Краткое описание работы нового функционала:
    а.	Добавление задачи или проекта.
        -	Нажать кнопку «Создать новую задачу / проект»;
        -	В открывшейся форме заполнить все поля (обязательно);
        -	В поле «Процент выполнения» необходимо вводить только целые числа;
        -	В поле «Исполнители» необходимо выбрать одного или нескольких исполнителей из выпадающего списка;
        -	В поле «Плановая дата выполнения» необходимо выбрать плановую дату исполнения задачи;
        -	Нажать кнопку «сохранить». Окно закроется и произойдет обновление страницы;
        -	Если необходимо очистить содержимое формы, то необходимо нажать на кнопку «очистить»;
        -	Для того чтобы закрыть форму без внесения изменений необходимо нажать на кнопку «закрыть».
    б.	Редактирование задачи или проекта
        -	Нажать кнопку «Редактировать задачу / проект»;
        -   В открывшейся форме в поле «Выберите задачу / проект для редактирования» необходимо выбрать название задачи или проекта. После выбора все остальные поля в форме буду заполнены текущими данными;
        -	Необходимо скорректировать одно или несколько полей, и нажать кнопку сохранить. Окно закроется и произойдет обновление страницы;
        -	Для того чтобы закрыть форму без внесения изменений необходимо нажать на кнопку «закрыть».

Важное замечание!
Название проекта или задачи изменить нельзя!
Остальные поля (исполнитель, процент выполнения, описание, плановая дата исполнения) можно редактировать.
В поле «Процент исполнения» можно вводить только целые числа.

2. Результат редактирования задач/проектов записываются в лог-файлы;
3. Удалена информация о среднем времени решения заявок;
4. Рефакторинг кода.
